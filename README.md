# Repuesto1
El proyecto consiste en el desarrollo de un sistema de escritorio en Windows Forms orientado a la gestión de una ferretería. Este sistema permitirá administrar de manera eficiente las operaciones principales del negocio, tales como el registro de productos, control de inventario, gestión de clientes y proveedores, así como la facturación de ventas.
El sistema contará con una interfaz amigable que incluirá un menú principal con accesos directos a las funciones más utilizadas, facilitando la navegación del usuario.
