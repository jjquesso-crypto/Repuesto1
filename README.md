<h1>Repuesto</h1>
<h3>Sistema de Gestión para Ferreterías y Tiendas de Repuestos</h3> </div>
Repuesto es un sistema de escritorio desarrollado en Windows Forms con C# y SQL Server, diseñado para optimizar la gestión administrativa y operativa de una ferretería o tienda de repuestos.
La aplicación permite controlar de manera eficiente los procesos fundamentales del negocio, incluyendo el registro y mantenimiento de productos, control de inventario, gestión de clientes y proveedores, facturación de ventas, compras, reportes y administración de usuarios con distintos roles y permisos.
El sistema cuenta con una interfaz intuitiva y amigable, organizada mediante un menú principal con accesos directos a las funcionalidades más utilizadas. Esto facilita la navegación del usuario y agiliza las tareas diarias, permitiendo un mejor control de las existencias, las transacciones comerciales y la información general del negocio.
